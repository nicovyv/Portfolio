Proyectos

Back Office

Proyecto final de la materia Programación 3 (UTN FRGP). Desarrollo en equipo un sistema Back Office para la gestión integral de un comercio. La aplicación permite gestionar clientes, productos, proveedores, ventas y compras con control de stock, cálculo automático de precios y validación por stock mínimo. Implementa login seguro con control de acceso por roles. Se utilizo ASP.NET Web Forms, C# y SQL Server, diseñando la base de datos desde cero.

Catálogo

Proyecto realizado para la materia Programación III de la Tecnicatura Universitaria en Programación (UTN). Consiste en una aplicación de escritorio desarrollada con WinForms y .NET Framework en C#, conectada a una base de datos SQL. La aplicación permite gestionar los artículos de un catálogo comercial, incluyendo el listado general, la búsqueda por distintos criterios, el alta, modificación y eliminación de artículos, así como la visualización detallada de cada uno.
